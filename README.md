# BoostupXI Compiler

This is a compiler API for the BoostupXI website.

## Results

| Status | Description             |
| ------ | ----------------------- |
| P      | The test passed.        |
| F      | The test failed.        |
| T      | The test timed out.     |
| E      | The test errored.       |
| R      | The test run timed out. |
| O      | The test out of buffer. |
| -      | The test was not run.   |
