# BoostupXI Compiler

This is a compiler API for the BoostupXI website.

## Results

| Status | Description             |
| ------ | ----------------------- |
| P      | The test passed.        |
| E      | The test errored.       |
| T      | The test timed out.     |
| R      | The test run timed out. |
| O      | The test out of buffer. |
| -      | The test was not run.   |
| C      | Created File Error      |
| L      | Library Ban             |
| W      | Testcase Error          |
| B      | Build Error             |
